#What is this?

This project allows a user to control a Sony Bravia TV with Amazon Alexa voice commands.

There are four main components

- Custom Amazon Alexa Skill 'alexa_config'
- AWS Lambda function
- AWS IOT Setup (*No Code Required*)
- Python MQTT Client
